# Snake Game

이 프로젝트는 간단한 웹 기반 스네이크 게임입니다.

- **제작 및 업로드:** Helpy chat과 GitHub MCP를 활용하여 자동으로 생성 및 업로드되었습니다.
- **구성 파일:**
  - `index.html` : 게임 실행을 위한 메인 HTML 파일
  - `snake.js` : 게임의 핵심 로직이 담긴 자바스크립트 파일

## GitHub Pages로 배포하는 방법

1. **저장소에서 [Settings] 클릭**
2. **왼쪽 메뉴에서 "Pages" 선택**
3. **"Branch"에서 `main`(또는 master) 브랜치와 `/ (root)` 폴더 선택**
4. **"Save" 버튼 클릭**
5. 잠시 기다리면, 상단에 배포된 링크가 생성됩니다. (예: `https://사용자명.github.io/저장소명/`)

> 이 과정을 거치면 웹 브라우저에서 바로 게임을 플레이할 수 있습니다!
